# 市場関係者の予想

## 日経（今週の市場) [リンク](https://www.nikkei.com/article/DGXZQOUB198NF0Z10C22A8000000/)
- (ジャクソンホール会議)市場が期待する利上げ鈍化をけん制する発言があれば、利益確定売りが上値を抑えかねない。
- 23日に8月の購買担当者景気指数（PMI、速報値）：現在の相場は「景気減速懸念」より「金融引き締めの鈍化期待」に反応しやすい。「よほど悪い結果でなければ、かえって好材料視される」
- 今週の米債券市場では長期金利に上昇（債券価格は下落）圧力か。米物価指標にはピークアウトの兆しがあるが、FRB高官は市場の利上げ観測後退をけん制する発言を繰り返している
- 先週は米原油先物（期近）が一時、半年ぶりの安値。PMIが予想以上に悪化すれば弱材料になるか。一方、ロシア産を中心に供給制約は解消しておらず、下値も限られて一進一退の値動きとの予想も

- 22－26日の米国株式市場は様子見か。ジャクソンホールでのFRB議長講演で大幅利上げの継続が予想されている9月FOMCのヒントを探る。注目度は高く、週を通じて動きにくい展開

## 東洋証券 [リンク](https://www.toyo-sec.co.jp/lineup/usa_stocks/forecast/220818_7566.html)
- 過熱感があるものの、需給環境が相場下支えか
- 米国株のバリュエーションを見ると翌4四半期の予想EPSを対象とした予想PERは既に18倍台に入っており、20/2/14の19.4倍や17/12/15の18.8倍が意識される水準と考える。今後も利上げが予想される中、バリュエーションの更なる拡大は期待しにくいと予想され、上値抑制材料
- 会議前は様子見ムード
- ズームビデオコミュニケーションズ（ZM、8/22）やセールスフォース（CRM、8/24）、ワークデイ（WDAY、8/25）などが決算発表を予定。安値から2～3割程度戻っており（終値基準、8/16時点）、戻りが正当化される内容か注目
- 米国株の投機筋は戻り相場について来られておらずS&P500のe-miniの非商業部門のネットポジションは約24万枚の売り越し（8/9時点）となっている。現物の空売りのポジションも増加したとの報道もある。押し目では買戻し需要が相場を下支えか。

## Investopedia [リンク](https://www.toyo-sec.co.jp/lineup/usa_stocks/forecast/220818_7566.html)
- Key Inflation Measure to be Released (PCE) : core PCE (energy costと食品を除いたPCE)の上昇率は4.8% (6月) -> 4.7% (7月)へとやや緩和される市場予想

## charles shwab [リンク](https://www.schwab.com/learn/story/weekly-traders-outlook)

- 決算発表した全体の76%の企業が市場の予想EPSを上回り、63%が予想収益を上回った
- 市場予想は上回ったが過去2年半の平均(80%と70%)と比較すると、緩やかに減速傾向
- 利益は+8.2%で予想の+4.1%を上回った。収益は+14.1%で予想の10.1%を上回った。(Q1は+7.6%と+13.1%だった)

- S&P500は200日平均線に押し戻され、新たなbull market入りとはならなかったが、ここ8週間で15%以上上昇している
